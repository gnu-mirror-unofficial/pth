Multithreading Libraries

The following is a collection of references to all known multithreading libraries for Unix operating systems.
  • Pth - GNU Portable Threads
    http://www.gnu.org/software/pth/
    ftp://ftp.gnu.org/gnu/pth/
    Ralf S. Engelschall, 1999-2000

  • CMU LWP - CMU Lightweight Processes
    ftp://ftp.coda.cs.cmu.edu/pub/coda/src/
    Larry Raper et all, 1984-2000

  • FSU Pthreads - FSU/PART POSIX Threads
    http://www.cs.fsu.edu/~mueller/pthreads/
    ftp://ftp.cs.fsu.edu/pub/PART/PTHREADS/
    Frank Mueller, 1992-1999

  • ST - SGI State Threads Library
    http://oss.sgi.com/projects/state-threads/
    Silicon Graphics, 2000.

  • NSPR - Netscape Portable Runtime
    http://www.mozilla.org/docs/refList/refNSPR/
    Netscape Corporation, Mozilla Project, 1998-2000.

  • pmp - Patched MIT POSIX Threads
    http://www.humanfactor.com/pthreads/mit-pthreads.html
    sdybiec@humanfactor.com, 1997-1998

  • MIT pthreads - MIT POSIX Threads
    http://www.mit.edu/~proven/pthreads.html
    ftp://sipb.mit.edu/pub/pthreads/
    Chris Provenzano, 1993-1996

  • PTL - Portable Thread Library
    http://www.media.osaka-cu.ac.jp/~k-abe/PTL/
    ftp://ftp.media.osaka-cu.ac.jp/pub/PTL/
    Kota Abe, 1997-1999

  • PTL - Portable Thread Library
    http://www-sor.inria.fr/~bruncott/PTL/
    ftp://ftp.inria.fr/INRIA/Projects/SOR/misc/thread-lib/
    Georges Brun-Cottan, 1996-1997

  • LinuxThreads - Linux POSIX Threads
    http://pauillac.inria.fr/~xleroy/linuxthreads/
    Xavier Leroy, 1997-1999

  • uthread - FreeBSD User-Land Threads
    ftp://ftp.freebsd.org/pub/FreeBSD/FreeBSD-current/src/lib/libc_r/uthread/
    John Birrell, 1998-1999

  • Cthreads - A parallel programming library
    http://www.cc.gatech.edu/systems/projects/Cthreads/
    Greg S. Eisenhauer, 1991-1997

  • PCThreads - POSIX.1c and DCE Threads For Linux
    http://members.aa.net/~mtp/PCthreads.html
    Michael T. Peterson, 1996

  • OpenThreads - Open Implementation of Lightweight Threads
    http://www.cs.uwyo.edu/~haines/research/ot/
    Matthew D. Haines, 1996-1997

  • RT++ - Higher Order Threads for C++
    http://www.risc.uni-linz.ac.at/software/rt++/
    ftp://ftp.risc.uni-linz.ac.at/pub/parlab/rt++/
    Wolfgang Schreiner, 1996-1997

  • pthreads-win32 - POSIX Threads for Win32
    http://sourceware.cygnus.com/pthreads-win32/
    Cygnus, 1998-1999

  • rsthreads - Really Simple Threads
    ftp://ftp.ai.mit.edu/pub/rst/
    Robert S. Thau, 1996

  • bb_threads - Bare-Bones Threads
    ftp://caliban.physics.utoronto.ca/pub/linux/
    Christopher Neufeld, 1996

  • jkthread - Simple Kernel Threads for Linux
    http://www.jdkoftinoff.com/linux3.html
    Jeff Koftinoff, 1998

  • NThreads - Threads for Numerical Applications
    ftp://ftp.tu-chemnitz.de/pub/Local/informatik/rechnerarchitektur/nthreads/
    Thomas Radke, 1997

  • RexThreads - Light-weight Processes for Rex
    ftp://ftp.inria.fr/prog/threads/
    Stephen Crane, 1993
